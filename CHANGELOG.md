## [1.0.0] — 2024-09-01 · Hackathon Release

### Added
- **Scam Detection** — real-time analysis of messages, emails, SMS
- **Link Analyzer** — URL and domain threat assessment
- **Social Manipulation Detector** — urgency, impersonation, romance scam detection
- **Risk Scoring** — continuous 0–100 score with 5-tier classification
- **Scan History** — persistent log with replay and export
- **JSON Export** — download full scan report
- **Hermes-4-70B integration** — via OpenRouter API
- **Zustand store** with localStorage persistence
- **Framer Motion** animations throughout
- **Vitest** unit + integration test suite
- **Playwright** E2E test suite
- **GitHub Actions** CI/CD pipeline
- **Docker** multi-stage build with nginx
- Full **TypeScript** coverage
- **Rate limiting** — 20 req/min client-side throttle
